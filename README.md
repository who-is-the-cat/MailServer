# Конфигурация для SOGo (взята из сборки ZEG)

## 1. Dovecot 

1) dovecot.conf переместить в /etc/dovecot/
2) Все конфигурационные файлы в /protocols.d/* переместить 
   в /usr/share/dovecot/protocols.d/
